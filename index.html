Welcome to our GitHub Notes Page

As there seamed to be so many enthusiastic Note Taking volunteers at our last meeting I thought I would make the job a bit more appealing by hosting our notes on GitHub. The plan is that each week someone who hasn't done it before will be the designated Note Taker for that week. We will of course get you setup but it will mean using Git and GitHub and writing the notes in either Markdown or HTML.
